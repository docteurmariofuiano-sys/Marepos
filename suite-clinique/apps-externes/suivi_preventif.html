Outil d’aide au suivi préventif de la patientèle médecin traitant

Importez vos exports CNAM pour identifier rapidement les patients nécessitant une vérification, un rappel ou une action de prévention.

L’outil prépare les messages à copier dans votre logiciel métier et les scripts d’appel pour votre secrétariat.

Aide organisationnelle.
Les actions proposées doivent être validées par le médecin selon le dossier clinique du patient. Aucun message n’est envoyé automatiquement.

Import des fichiers CNAM

Attention : ces données sont confidentielles. Elles doivent être conservées uniquement le temps nécessaire à l’accompagnement de la patientèle, stockées dans un espace sécurisé, puis supprimées une fois les actions effectuées.

Déposez vos fichiers PDF, CSV ou Excel

Les fichiers sont lus localement dans le navigateur. Les fichiers importés ne sont pas conservés automatiquement.

Contrôles qualité

Documentation courte

Importez un export CNAM, vérifiez l’indicateur détecté, puis traitez les patients affichés dans la liste de travail. Les boutons copient les textes préparés ; l’envoi reste manuel dans votre logiciel métier. Utilisez l’export Excel, CSV ou PDF pour conserver une trace sécurisée si nécessaire.

Confidentialité : le NIR est masqué par défaut. Les identifiants insuffisamment masqués déclenchent une alerte et sont automatiquement masqués dans l’interface et les exports.

Modèles de messages et scripts

Les modèles intégrés sont neutres, courts, sans diagnostic, sans NIR et adaptés au copier-coller. Ils peuvent être personnalisés directement dans les champs générés avant copie si besoin.

Données confidentielles - usage réservé au médecin et à son équipe autorisée. Les fichiers importés doivent être supprimés après traitement. L’outil ne remplace pas l’analyse médicale du dossier patient. Aucune donnée ne doit être transmise à des tiers non autorisés.
